Shortcode for displaying specific number of recent products per vendor.

== Description ==

This WC Marketplace plugin (https://wordpress.org/plugins/dc-woocommerce-multi-vendor/) extension enables [wcmp_recent_products_per_vendor] shortcode for displaying specific number of recent products per vendor (default is 2) . You can optionally use it with "per_vendor" and "total" parameters, for example [wcmp_recent_products_per_vendor per_vendor="5" total="20"]

This plugin is developed by request: https://wordpress.org/support/topic/display-recent-products-but-only-2-per-vendor/#post-11496330

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WCMP Recent Products Per Vendor Shortcode'
3. When you find it, click 'install', and after that, click 'activate'
4. Just use [wcmp_recent_products_per_vendor] shortcode, along with "per_vendor" and "total" parameters if needed.

= From WordPress.org =

1. Download 'WCMP Recent Products Per Vendor Shortcode'.
2. Upload the 'WCMP Recent Products Per Vendor Shortcode' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. When you upload it, click 'install', and after that, click 'activate'
4. Just use [wcmp_recent_products_per_vendor] shortcode, along with "per_vendor" and "total" parameters if needed.
